# 市集卡片日期顯示測試

## 修改內容

### 1. 市集頁面 (`/markets`)
- **顯示邏輯**: 完整顯示所有日期，支援換行
- **多選日期**: 使用 `formatDateRanges()` 智能合併連續日期
  - 例如: `2024/02/15~17, 2024/02/22~23`
- **連續日期**: 顯示 `startDate - endDate`
  - 例如: `2024/02/15 - 2024/02/17`
- **單日**: 顯示單一日期
  - 例如: `2024/02/15`

### 2. 首頁 (`/`)
- **顯示邏輯**: 只顯示當週的日期（簡化顯示）
- **多選日期**: 
  - 使用 `filterCurrentWeekDates()` 過濾出當週日期
  - 再用 `formatDateRanges()` 格式化
  - 如果當週沒有日期，顯示最近的一個日期
- **連續日期**: 顯示完整範圍（不過濾）
  - 例如: `2024/02/15 - 2024/02/17`

### 3. 市集詳情頁 (`/markets/[id]`)
- **顯示邏輯**: Header 完整顯示所有日期，支援換行
- **多選日期**: 使用 `formatDateRanges()` 智能合併
- **連續日期**: 顯示 `startDate - endDate`
- **單日**: 顯示單一日期

## 新增工具函數

### `formatDateRanges(dates: string[])`
智能合併連續日期為範圍顯示：
```typescript
// 輸入: ['2024-02-15', '2024-02-16', '2024-02-17', '2024-02-22', '2024-02-23']
// 輸出: '2024/02/15~17, 2024/02/22~23'

// 輸入: ['2024-02-15', '2024-02-17', '2024-02-20']
// 輸出: '2024/02/15, 2024/02/17, 2024/02/20'
```

### `filterCurrentWeekDates(dates: string[])`
過濾出當週的日期（週日到週六）：
```typescript
// 假設今天是 2024-02-16 (週五)
// 本週範圍: 2024-02-11 (週日) ~ 2024-02-17 (週六)

// 輸入: ['2024-02-10', '2024-02-15', '2024-02-16', '2024-02-20']
// 輸出: ['2024-02-15', '2024-02-16']
```

## 測試場景

### 場景 1: 單日市集
- **dates**: `undefined` 或 `[]`
- **startDate**: `2024-02-15`
- **endDate**: `2024-02-15`
- **市集頁面**: `2024/02/15`
- **首頁**: `2024/02/15`
- **詳情頁**: `2024/02/15`

### 場景 2: 連續日期市集
- **dates**: `undefined` 或 `[]`
- **startDate**: `2024-02-15`
- **endDate**: `2024-02-17`
- **市集頁面**: `2024/02/15 - 2024/02/17`
- **首頁**: `2024/02/15 - 2024/02/17`
- **詳情頁**: `2024/02/15 - 2024/02/17`

### 場景 3: 多選連續日期
- **dates**: `['2024-02-15', '2024-02-16', '2024-02-17']`
- **市集頁面**: `2024/02/15~17`
- **首頁**: 
  - 如果在當週: `2024/02/15~17`
  - 如果不在當週: `2024/02/15`（顯示第一個日期）
- **詳情頁**: `2024/02/15~17`

### 場景 4: 多選不連續日期
- **dates**: `['2024-02-15', '2024-02-17', '2024-02-20']`
- **市集頁面**: `2024/02/15, 2024/02/17, 2024/02/20`（可換行）
- **首頁**: 
  - 如果在當週: `2024/02/15, 2024/02/17, 2024/02/20`
  - 如果不在當週: `2024/02/15`
- **詳情頁**: `2024/02/15, 2024/02/17, 2024/02/20`（可換行）

### 場景 5: 多選混合日期
- **dates**: `['2024-02-15', '2024-02-16', '2024-02-17', '2024-02-22', '2024-02-23']`
- **市集頁面**: `2024/02/15~17, 2024/02/22~23`（可換行）
- **首頁**: 
  - 如果在當週: 過濾後的日期範圍
  - 如果不在當週: `2024/02/15`
- **詳情頁**: `2024/02/15~17, 2024/02/22~23`（可換行）

## UI 改進

### 市集頁面卡片
- 日期區域使用 `flex-wrap` 支援換行
- Icon 使用 `flex-shrink-0` 防止壓縮
- 文字使用 `flex-1` 自動換行

### 市集詳情頁 Header
- 日期和地點分兩行顯示
- 日期區域支援換行
- Icon 使用 `flex-shrink-0` 和 `mt-0.5` 對齊

## 向後兼容

所有修改都保持向後兼容：
- 如果 `market.dates` 不存在或為空，自動降級到 `startDate/endDate` 邏輯
- 舊的連續日期市集仍然正常顯示
- 新的多選日期市集使用智能合併顯示
