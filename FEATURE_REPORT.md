## 🎯 專案概述

**Market Pulse** 是一個專為市集攤販設計的離線優先數位管理系統，幫助攤販在市集現場快速記錄互動、管理商品、追蹤銷售，並提供深入的數據分析。

### 核心價值
- ✅ **完全離線運作** - 使用 IndexedDB 本地儲存，無需網路連線
- ✅ **快速互動記錄** - 一鍵記錄顧客互動（詢問、試吃、拍照等）
- ✅ **即時數據統計** - 營業中即時顯示收入、成交數、轉換率
- ✅ **靈活庫存管理** - 支援實體商品庫存追蹤與無限庫存模式
- ✅ **深度數據分析** - 多維度圖表分析營業表現

## 🚀 核心功能

### 1. 市集管理
- 新增/編輯/刪除市集
- 多日期市集支援（跨日活動）
- 市集狀態流轉（已報名 → 已錄取 → 已繳費 → 營業中 → 已完成）
- 詳細成本追蹤（攤位費、租金、抽成等）
- 時間軸管理（提前進場、報到、營業時間）

### 2. 商品管理
- 商品 CRUD 操作
- 7 種商品分類（手作、食品、飾品、服飾、藝術、文具、其他）
- 靈活庫存系統：
  - 實體商品：追蹤庫存數量
  - 服務/訂製：無限庫存模式
- 商品搜尋與分類篩選
- 視覺識別（圖示、顏色）

### 3. 營業中功能
- **快速互動按鈕**（可自訂）
  - 預設：詢問、試吃、拍照
  - 即時顯示互動次數
  - 一鍵記錄顧客互動
- **快速成交**
  - 直接輸入金額成交（無需選商品）
  - 適合小額交易或服務
- **購物車系統**
  - 多商品加入購物車
  - 數量調整
  - 即時計算總額
  - 支援多種支付方式
- **即時指標**
  - 今日收入
  - 成交數
  - 轉換率
  - 平均客單價

### 4. 數據分析
- **日期範圍篩選**（今日、本週、本月、全選）
- **關鍵指標卡片**
  - 市集總數
  - 總收入
  - 總淨利潤
  - 平均轉換率
- **視覺化圖表**
  - 營收趨勢圖（折線圖）
  - 互動與成交對比圖（柱狀圖）
  - 轉換漏斗圖
  - 成本分析圖（圓餅圖）
- **市集明細列表**
  - 每場市集的詳細數據
  - 互動類型統計
  - 成本與利潤分析

### 5. 設定功能
- **快速互動按鈕自訂**
  - 自訂按鈕文字（最多 4 字）
  - 自訂 Emoji 圖示
  - 即時預覽效果
  - 手風琴式折疊設計（節省空間）

## 📱 頁面功能詳解

### 1. 首頁 (`/`)
**功能**：
- 本月概覽（市集場次、總收入、成交數）
- 今日市集快速進入
- 即將到來的市集列表
- 快速導航到各功能頁面

**特色**：
- 自動篩選當日與未來市集
- 一鍵進入營業模式
- 視覺化月度統計

---

### 2. 市集頁 (`/markets`)
**功能**：
- 市集列表（全部、已報名、未繳費、如期舉行）
- 新增市集（浮動按鈕）
- 市集卡片（顯示狀態、日期、地點、收入）
- 點擊進入市集詳情

**Tab 分類**：
- **全部**：所有市集
- **已報名**：status = 'registered'
- **未繳費**：status = 'accepted'
- **如期舉行**：status = 'paid' 或 'ongoing'

**市集卡片資訊**：
- 市集名稱與地點
- 日期範圍（支援多日）
- 狀態標籤（顏色編碼）
- 收入統計（已完成市集）

---

### 3. 市集詳情頁 (`/markets/[id]`)
**功能**：
- 市集資訊展示
- 狀態管理（開始營業、結束營業）
- 營業中功能：
  - 快速互動按鈕
  - 快速成交
  - 購物車
  - 即時指標
- 市集數據統計
- 成本明細
- 編輯/刪除市集

**營業流程**：
1. 點擊「開始營業」→ 狀態變為 `ongoing`
2. 使用快速互動按鈕記錄顧客互動
3. 使用購物車或快速成交完成交易
4. 點擊「結束營業」→ 狀態變為 `completed`

**即時指標**：
- 今日收入（即時更新）
- 成交數（即時更新）
- 轉換率（互動數 / 成交數）
- 平均客單價

---

### 4. 商品頁 (`/products`)

- 商品列表（網格佈局）

- 新增商品（浮動按鈕）

**商品卡片**：

## 🗄️ 資料庫設計

### 表結構

#### 1. `events` 表（事件日誌）
**用途**：不可變的事件記錄，所有操作都記錄為事件

**欄位**：
- `id` (主鍵，自動遞增)
- `type` (事件類型)
- `payload` (事件資料)
- `timestamp` (時間戳)
- `metadata` (元數據)

**事件類型**：
- `market_created` - 市集建立
- `market_status_changed` - 市集狀態變更
- `market_started` - 市集開始營業
- `market_ended` - 市集結束營業
- `product_created` - 商品建立
- `product_updated` - 商品更新
- `product_deleted` - 商品刪除
- `interaction_recorded` - 互動記錄
- `deal_closed` - 成交記錄

#### 2. `markets` 表（市集快照）
**用途**：市集當前狀態的快照

**主要欄位**：
- `id` (主鍵)
- `name` (市集名稱)
- `location` (地點)
- `startDate` / `endDate` (日期範圍)
- `status` (狀態)
- `boothCost` (攤位成本)
- `commissionRate` (抽成比例)
- `totalRevenue` (總收入)
- `totalProfit` (總利潤)
- `totalInteractions` (總互動數)
- `totalDeals` (總成交數)

#### 3. `products` 表（商品清單）
**用途**：商品資訊

**主要欄位**：
- `id` (主鍵)
- `name` (商品名稱)
- `category` (分類)
- `price` (售價)
- `cost` (成本)
- `stock` (庫存)
- `unlimitedStock` (無限庫存標記)
- `isActive` (是否啟用)

#### 4. `dailyStats` 表（每日統計）
**用途**：快速查詢每日數據

**主要欄位**：
- `date` (日期，主鍵)
- `marketId` (關聯市集)
- `revenue` (收入)
- `dealCount` (成交數)

#### 5. `settings` 表（設定）
**用途**：用戶設定

**主要欄位**：
- `id` (固定為 1)
- `theme` (主題)
- `language` (語言)
- `defaultCurrency` (預設貨幣)

## 🎨 UI/UX 設計

### 設計原則
1. **離線優先** - 所有功能無需網路
2. **快速操作** - 一鍵完成常用操作
3. **即時反饋** - 操作立即顯示結果
4. **視覺清晰** - 大按鈕、清晰標籤
5. **行動優先** - 針對手機螢幕優化

### 視覺特色
- **大圓角設計** (1.5rem - 2rem)
- **漸層背景** (品牌色漸層)
- **卡片式佈局** (陰影與層次)
- **微互動動畫** (縮放、波紋、彈跳)
- **顏色編碼** (狀態、分類、指標)

### 互動設計
- **按鈕反饋**：
  - Hover：放大 (scale-105)
  - Active：縮小 (scale-95)
  - 點擊：波紋效果
- **+1 動畫**：互動記錄時顯示
- **Toast 通知**：操作成功/失敗提示
- **Loading 狀態**：資料載入時顯示

### 響應式設計
- **最大寬度**：max-w-lg (512px)
- **間距系統**：4px 基準 (gap-3, gap-4, gap-6)
- **字體大小**：text-xs ~ text-2xl
- **觸控友好**：按鈕最小 44x44px
