## 互動動畫

```

懸停狀態：shadow-lg (陰影加深)
過渡時間：transition-shadow
```

```css
@keyframes slide-up {
  from {
    transform: translateY(100%);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}

.animate-slide-up {
  animation: slide-up 0.3s ease-out;
}
```

```
正常：bg-[#7B9FA6]
懸停：bg-[#6A8E95] (顏色加深)
禁用：opacity-50 + cursor-not-allowed
過渡：transition-colors
```

## 設計原則

### 1. 極簡主義
- 去除不必要的裝飾
- 專注於內容本身
- 留白充足

### 2. 日系美學
- 柔和的色彩
- 大圓角設計
- 細膩的陰影

### 3. 視覺層次
- 使用色彩區分重要性
- 使用大小建立層次
- 使用間距分組元素

### 4. 一致性
- 統一的圓角尺寸
- 統一的間距系統
- 統一的色彩使用

### 5. 可讀性
- 充足的對比度
- 適當的字體大小
- 清晰的視覺引導
