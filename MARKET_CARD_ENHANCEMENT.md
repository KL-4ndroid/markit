# 市集排名卡片優化報告

## 📋 實作摘要

已成功為所有市集排名卡片加入日期顯示和點擊跳轉功能，讓用戶能清楚識別具體場次並快速查看詳情。

## ✅ 優化內容

### 1. 日期顯示邏輯

#### 智能日期格式化
針對不同的市集類型，顯示最合適的日期格式：

**單日市集**：
```
2024-03-15
```

**多選日期（單日）**：
```
2024-03-15
```

**多選日期（多日）**：
```
2024-03-15 等 3 天
```

**連續日期（同一天）**：
```
2024-03-15
```

**連續日期（多天）**：
```
2024-03-15 ~ 2024-03-17
```

#### 實作邏輯
```typescript
// 優先檢查 dates 陣列（多選日期）
if (market.dates && market.dates.length > 0) {
  if (market.dates.length === 1) {
    return market.dates[0];
  }
  return `${market.dates[0]} 等 ${market.dates.length} 天`;
}

// 降級：使用 startDate 和 endDate（連續日期）
if (market.startDate === market.endDate) {
  return market.startDate;
}
return `${market.startDate} ~ ${market.endDate}`;
```

### 3. 點擊跳轉功能

#### 已有功能（保持不變）
- ✅ MarketROICard - 已有 `onClick` 跳轉
- ✅ MarketAOVCard - 已有 `onClick` 跳轉

#### 新增功能
- ✅ QuadrantGrid - 為每個市集項目加入點擊跳轉

#### 實作方式

**MarketROICard & MarketAOVCard**：
```typescript
const handleClick = () => {
  router.push(`/markets/${market.id}`);
};

<div 
  onClick={handleClick}
  className="... cursor-pointer hover:bg-[#F0F0EE] transition-all ..."
>
```

**QuadrantGrid**：
```typescript
const router = useRouter();

<div
  onClick={() => router.push(`/markets/${market.id}`)}
  className="... cursor-pointer hover:bg-white transition-colors"
>
```

#### 視覺反饋
- ✅ `cursor-pointer` - 滑鼠指標變為手型
- ✅ `hover:bg-[#F0F0EE]` - Hover 時背景變淺
- ✅ `transition-all` / `transition-colors` - 平滑過渡動畫
- ✅ `hover:border-[#7B9FA6]/30` - Hover 時邊框加深（ROI/AOV 卡片）

## 💡 設計考量

### 為什麼使用 `•` 分隔符號？
- ✅ 視覺上輕盈，不會過於突兀
- ✅ 國際通用符號，易於理解
- ✅ 使用品牌色增加一致性

### 為什麼日期不截斷？
- ✅ 日期是關鍵識別資訊
- ✅ 地點可以從市集名稱推測，日期無法推測
- ✅ 使用 `flex-shrink-0` 確保日期完整顯示

### 為什麼象限網格使用兩行？
- ✅ 避免單行過於擁擠
- ✅ 保持視覺層次（名稱 > 日期）
- ✅ 使用較小字體和 emoji 降低視覺權重
