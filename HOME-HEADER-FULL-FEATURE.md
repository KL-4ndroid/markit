## 🎨 新功能說明

### 同步狀態按鈕

#### 圖示狀態
- 🔄 `Loader2`（旋轉動畫）- 同步中
- ✅ `CheckCircle` - 已同步
- ⚠️ `AlertCircle` - 同步失敗
- ☁️ `Cloud` - 閒置
- 📴 `CloudOff` - 離線/未登入

#### 交互功能
1. **點擊**：手動觸發同步（如果已登入且不在同步中）
2. **滑鼠懸停**：顯示詳細的同步狀態 Tooltip

#### Tooltip 內容
```
┌─────────────────────────────┐
│ 狀態      已同步 ✅          │
│ 網路      🟢 已連線          │
│ 最後同步   剛剛              │
│ 待同步     3 個事件          │
│ ─────────────────────────   │
│ [立即同步] 按鈕              │
└─────────────────────────────┘
```

---

### 使用者登入狀態按鈕

#### 圖示狀態
- 👤 `UserCircle`（實心）- 已登入
- 👤 `User`（空心）- 未登入

#### 已登入時的交互
1. **點擊**：顯示用戶選單
2. **選單內容**：
   - 顯示登入的 Email
   - 登出按鈕

#### 未登入時的交互
1. **點擊**：觸發登入對話框

## 🎯 用戶交互流程

### 查看同步狀態
1. 用戶看到同步圖示（動態變化）
2. 滑鼠懸停查看詳細狀態
3. 點擊手動觸發同步
4. Tooltip 中可以看到：
   - 當前狀態
   - 網路連線狀態
   - 最後同步時間
   - 待同步事件數量
   - 錯誤訊息（如果有）

### 登入/登出
1. **未登入時**：
   - 看到空心用戶圖示
   - 點擊觸發登入對話框
   
2. **已登入時**：
   - 看到實心用戶圖示
   - 點擊顯示用戶選單
   - 選單顯示 Email
   - 點擊「登出」按鈕登出

## 🔧 技術實現

### 狀態管理
```typescript
const [showSyncTooltip, setShowSyncTooltip] = useState(false);
const [showUserMenu, setShowUserMenu] = useState(false);
```

### Hooks 使用
```typescript
const { user, signOut, isConfigured } = useAuth();
const { status, lastSyncAt, pendingCount, error, sync, isOnline } = useSync({
  enabled: !!user && isConfigured,
});
```

### 事件處理
```typescript
// 手動同步
onClick={() => {
  if (user && status !== SyncStatusEnum.SYNCING) {
    sync();
  }
}}

// 登出
const handleSignOut = async () => {
  await signOut();
  toast.success('已登出');
  setShowUserMenu(false);
};

// 登入
const handleLogin = () => {
  const button = document.getElementById('auth-manager-login-trigger');
  if (button) {
    button.click();
  }
};
```

## ⚠️ 注意事項

### 1. 只在首頁有這些按鈕
- 其他頁面（市集、商品、分析、設定）沒有頂部導航
- 如果需要在其他頁面也顯示，可以：
  - 在每個頁面的 Header 添加
  - 或者恢復 TopNavigation

### 2. 背景點擊關閉
- Tooltip 和選單都有背景遮罩
- 點擊背景自動關閉

### 3. 未配置 Supabase 時
- 按鈕不顯示（`isConfigured` 檢查）
- 避免不必要的 UI 元素
