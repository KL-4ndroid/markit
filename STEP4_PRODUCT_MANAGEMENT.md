## 🔄 事件整合

所有商品操作都已整合事件溯源架構：

### 觸發的事件類型：

1. **`product_created`** - 建立商品
   ```typescript
   {
     name: string;
     category: ProductCategory;
     price: number;
     cost?: number;
     stock?: number;
     description?: string;
   }
   ```

2. **`product_updated`** - 更新商品
   ```typescript
   {
     productId: number;
     updates: Partial<Product>;
   }
   ```

3. **`product_deleted`** - 刪除商品（軟刪除）
   ```typescript
   {
     productId: number;
   }
   ```

### 事件處理器（已存在於 `lib/db/events.ts`）：

- ✅ `handleProductCreated` - 建立新商品記錄
- ✅ `handleProductUpdated` - 更新商品資料
- ✅ `handleProductDeleted` - 標記商品為停用
