## 🔧 技術實施

### CSS 變量系統
```css
:root {
  /* 配色 */
  --mist-blue: #7B9FA6;
  --warm-wood: #D4A574;
  --soft-pink: #F5E6E8;
  --soft-green: #E8F3E8;
  --soft-yellow: #FFF8E7;
  --warm-white: #FAFAF8;
  
  /* 圓角 */
  --radius-lg: 1.5rem;
  --radius-md: 1.25rem;
  --radius-sm: 0.75rem;
  
  /* 陰影 */
  --shadow-color: 195 25% 57%;
}
```

### 自定義樣式類
```css
.card-japanese { /* 日系卡片 */ }
.gradient-header { /* 漸變頭部 */ }
.stat-card { /* 統計卡片 */ }
.input-japanese { /* 日系輸入框 */ }
.loading-spinner { /* 載入動畫 */ }
```

### 常用 Tailwind 組合
```tsx
// 大卡片
"bg-white rounded-[1.5rem] shadow-lg shadow-[#7B9FA6]/10 p-6"

// 小卡片
"bg-white rounded-[1.25rem] shadow-md shadow-[#7B9FA6]/5 p-4"

// 主按鈕
"bg-[#7B9FA6] hover:bg-[#7B9FA6]/90 text-white px-4 py-2 rounded-xl font-medium"

// 漸變頭部
"gradient-header pt-12 pb-8 px-6 rounded-b-[2rem]"
```
