# 🎉 PWA 安裝提示優化完成

## ✅ 優化內容

### 原本的行為
- ❌ 每次開啟都會彈出安裝提示
- ❌ 關閉後 7 天內不顯示，7 天後又彈出
- ❌ 對用戶造成重複打擾

### 優化後的行為

#### 1️⃣ 第一次訪問
- ✅ 延遲 3 秒後自動顯示安裝提示
- ✅ 用戶可選擇「立即安裝」或「稍後」
- ✅ 記錄已顯示過，不再重複

#### 2️⃣ 第二次及之後訪問
- ✅ 不再自動彈出提示
- ✅ 安裝選項移至「設置」頁面
- ✅ 用戶可自行決定何時安裝

#### 3️⃣ 已安裝後
- ✅ 設置頁面的安裝按鈕自動消失
- ✅ 不再顯示任何安裝提示

   - 檢測是否已安裝

   - 使用 `localStorage` 記錄狀態

2. **`app/settings/page.tsx`**
   - 整合 `PWAInstallButton` 組件
   - 顯示在設置頁面頂部

## 📱 使用流程

### 第一次訪問
```
開啟網站 → 等待 3 秒 → 自動顯示安裝提示
   ↓
選擇「立即安裝」→ 完成安裝
   或
選擇「稍後」→ 關閉提示
```

### 第二次訪問
```
開啟網站 → 不再彈出提示
   ↓
前往「設置」頁面 → 看到安裝按鈕
   ↓
點擊按鈕 → 顯示安裝引導/直接安裝
```

### 已安裝
```
從主畫面啟動 App
   ↓
設置頁面不顯示安裝按鈕
```

## 🧪 測試方法

### 測試第一次訪問
```javascript
// 在 Console 執行
localStorage.removeItem('pwa-install-first-shown');
location.reload();
// 等待 3 秒，應該會彈出安裝提示
```

### 測試第二次訪問
```
1. 第一次訪問後，直接重新整理頁面
2. 應該不會再彈出安裝提示
3. 前往設置頁面
4. 應該看到「安裝市集誌到主畫面」按鈕
```

### 測試已安裝狀態
```
1. 完成 PWA 安裝
2. 從主畫面啟動 App
3. 前往設置頁面
4. 安裝按鈕應該消失
```
