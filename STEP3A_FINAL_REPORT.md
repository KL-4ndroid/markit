## 💡 技術亮點

### 1. 事件溯源架構
```typescript
// 用戶提交表單
await createMarket(formData);

// 自動觸發事件溯源：
// 1. 建立 Event { type: 'market_created', payload: {...} }
// 2. 寫入 events 表（不可變記錄）
// 3. 執行事件處理器
// 4. 寫入 markets 表（快照）
// 5. 列表自動更新（useLiveQuery）
```

### 2. 響應式資料
```typescript
// 使用 useLiveQuery 實現即時更新
const markets = useMarkets({ orderBy: 'date', order: 'desc' });

// 當資料變更時，組件自動重新渲染
// 無需手動刷新或狀態管理
```

```typescript
// 完整的 TypeScript 型別定義
interface MarketCreatedPayload {

  // ...
}

// 編譯時錯誤檢查
// IDE 自動完成
```

## 🔍 資料庫驗證

### IndexedDB 結構
```
MarketPulseDB
├── events
│   └── { id: 1, type: 'market_created', payload: {...}, timestamp: ... }
├── markets
│   └── { id: 1, name: '...', status: 'registered', ... }
├── products
├── dailyStats
└── settings
```

### 事件記錄
每次新增市集都會記錄：
- 事件類型：`market_created`
- 事件時間：精確到毫秒
- 事件資料：完整的市集資訊
- 元數據：版本號等

### 快照更新
市集快照包含：
- 基本資訊（名稱、地點、日期）
- 財務資訊（報名費、攤位成本）
- 統計資訊（收入、利潤、成交數）
- 時間戳（建立時間、更新時間）
