## 🧪 測試場景

### 場景 1：未登入用戶

1. 開啟設定頁面
2. 修改快速互動按鈕
3. 點擊「儲存設定」
4. ✅ 應顯示：「設定已儲存到本地」
5. ✅ localStorage 應有資料
6. ✅ Supabase 不應有資料

### 場景 2：已登入用戶 - 首次登入

1. 登入帳號
2. ✅ 應自動初始化預設設定到雲端
3. ✅ 應顯示：「用戶設定已初始化」
4. 開啟設定頁面
5. 修改快速互動按鈕
6. 點擊「儲存設定」
7. ✅ 應顯示：「設定已儲存並同步到雲端」
8. ✅ localStorage 和 Supabase 都應有資料

### 場景 3：已登入用戶 - 跨設備同步

1. 在設備 A 登入並修改設定
2. 登出
3. 在設備 B 登入相同帳號
4. ✅ 應自動拉取設備 A 的設定
5. ✅ 應顯示：「已從雲端同步用戶設定」
6. ✅ 設定應與設備 A 一致

### 場景 4：網路錯誤處理

1. 登入帳號
2. 斷開網路
3. 修改設定並保存
4. ✅ 應顯示：「設定已儲存到本地」（不報錯）
5. ✅ localStorage 應有資料
6. 恢復網路
7. 再次保存
8. ✅ 應顯示：「設定已儲存並同步到雲端」

## 📝 注意事項

1. **離線優先**：即使雲端同步失敗，本地儲存仍然成功
2. **自動同步**：登入時自動拉取，無需手動操作
3. **錯誤處理**：所有雲端操作都有 try-catch 保護
4. **用戶體驗**：明確的成功/失敗提示
