# 畫面閃爍問題修復文檔

## 🐛 **問題描述**

### 用戶報告的問題
1. **白屏閃爍**：點擊導航列時出現白屏
2. **骨架屏閃爍**：骨架屏顯示不穩定
3. **雙重閃爍**：完整內容滑入後，又閃爍一次白屏，然後才顯示完整內容
4. **發生時機**：點擊導航列時
5. **持續時間**：一瞬間
6. **影響設備**：電腦、手機

### 問題時間軸

```
原始流程（有問題）：
T+0ms    點擊導航
T+10ms   動畫開始（opacity: 0, translateX: 100%）  ← 白屏閃爍 1
T+100ms  內容開始載入
T+200ms  骨架屏顯示（延遲 200ms）                 ← 白屏閃爍 2
T+300ms  動畫結束（opacity: 1）
T+350ms  內容完全載入                              ← 白屏閃爍 3
T+400ms  最終顯示
```

---

## 🔍 **根本原因分析**

### 1. **動畫 opacity 設為 0**
```css
/* 問題代碼 */
@keyframes slideFromRight {
  from {
    transform: translateX(100%);
    opacity: 0;  /* ← 完全透明，造成白屏 */
  }
}
```

**影響**：
- ❌ 動畫開始時內容完全不可見
- ❌ 用戶看到 300ms 的白屏

---

### 2. **滑動距離過大**
```css
/* 問題代碼 */
transform: translateX(100%);  /* ← 完全移出螢幕 */
```

**影響**：
- ❌ 內容需要從螢幕外滑入
- ❌ 動畫時間過長

---

### 3. **骨架屏延遲顯示**
```css
/* 問題代碼 */
.skeleton-shimmer {
  animation-delay: 0.2s;  /* ← 延遲 200ms */
  opacity: 0;             /* ← 初始透明 */
}
```

**影響**：
- ❌ 前 200ms 完全看不到骨架屏
- ❌ 與頁面動畫疊加，造成雙重閃爍

---

### 4. **缺少背景色保護**
```typescript
/* 問題代碼 */
<div className="page-transition">
  {/* 沒有背景色 */}
</div>
```

**影響**：
- ❌ 內容載入前顯示白色背景
- ❌ 與動畫疊加造成閃爍

---

## ✅ **修復方案**

### 修復 1：優化動畫參數

**文件**：`app/globals.css`

**修改前**：
```css
@keyframes slideFromRight {
  from {
    transform: translateX(100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}
```

**修改後**：
```css
@keyframes slideFromRight {
  from {
    transform: translateX(30%);  /* 從 100% 改為 30% */
    opacity: 0.3;                 /* 從 0 改為 0.3 */
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}
```

**改進效果**：
- ✅ 內容始終部分可見（opacity: 0.3）
- ✅ 滑動距離減少 70%
- ✅ 動畫更流暢，不會完全白屏

---

### 修復 2：縮短動畫時長

**修改前**：
```css
.slide-from-right {
  animation: slideFromRight 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
```

**修改後**：
```css
.slide-from-right {
  animation: slideFromRight 0.25s cubic-bezier(0.4, 0, 0.2, 1);
}
```

**改進效果**：
- ✅ 從 300ms 縮短到 250ms
- ✅ 減少 50ms 的空白期
- ✅ 響應更快

---

### 修復 3：移除骨架屏延遲

**文件**：`app/globals.css`

**修改前**：
```css
.skeleton-shimmer {
  animation-delay: 0.2s;
  opacity: 0;
  animation-fill-mode: forwards;
}
```

**修改後**：
```css
.skeleton-shimmer {
  /* 移除 animation-delay */
  opacity: 1;  /* 立即顯示 */
  /* 移除 animation-fill-mode */
}
```

**改進效果**：
- ✅ 骨架屏立即顯示
- ✅ 消除 200ms 的空白期
- ✅ 與動畫同步

---

### 修復 4：添加背景色保護

**文件**：`app/globals.css`

**修改前**：
```css
.page-transition {
  will-change: transform, opacity;
  transform: translateZ(0);
  backface-visibility: hidden;
}
```

**修改後**：
```css
.page-transition {
  will-change: transform, opacity;
  transform: translateZ(0);
  backface-visibility: hidden;
  background-color: #FAFAF8;  /* 添加背景色 */
  min-height: 100vh;          /* 確保全屏 */
}
```

**改進效果**：
- ✅ 防止白屏
- ✅ 平滑過渡
- ✅ 視覺連續性

---

### 修復 5：優化淡入動畫

**文件**：`app/globals.css`

    opacity: 0;
  }

  }

**修改後**：
```css

  }

    opacity: 1;
  }

```

- ✅ 內容始終部分可見
- ✅ 減少閃爍感
- ✅ 更平滑的過渡

---

### 修復 6：同步重置時機

**文件**：`lib/navigation-context.tsx`

**修改前**：
```typescript
setTimeout(() => {
  setDirection('none');
}, 350); // 略長於動畫時間
```

**修改後**：
```typescript
setTimeout(() => {
  setDirection('none');
}, 300); // 與動畫時長同步（250ms + 50ms 緩衝）
```

**改進效果**：
- ✅ 狀態重置與動畫同步
- ✅ 避免時序混亂
- ✅ 減少閃爍
