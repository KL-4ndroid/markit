### 測試 3：建立測試數據 ⭐⭐

**目標：** 準備完整的測試數據

**步驟：**

#### 3.1 建立市集
1. 進入「市集」頁面
2. 點擊「+」按鈕
3. 填寫表單：
   ```
   市集名稱：測試市集 2026
   地點：台北華山
   開始日期：今天
   結束日期：今天
   時間：10:00 - 18:00
   報名費：100
   攤位成本：500
   ```
4. 點擊「建立市集」

#### 3.2 建立商品（至少 3 個不同分類）
1. 進入「商品」頁面
2. 建立商品 1：
   ```
   名稱：手工陶杯
   分類：手作 🧵
   售價：350
   成本：150
   庫存：10
   ```
3. 建立商品 2：
   ```
   名稱：手工餅乾
   分類：食品 🍰
   售價：120
   成本：50
   庫存：20
   ```
4. 建立商品 3：
   ```
   名稱：手工耳環
   分類：飾品 💎
   售價：280
   成本：100
   庫存：15
   ```

#### 3.3 記錄互動和交易
1. 進入「測試市集 2026」詳情頁
2. 點擊「開始營業」
3. 記錄互動：
   - 點擊「摸摸」5 次
   - 點擊「詢問」3 次
4. 記錄交易：
   - 交易 1：手工陶杯 x 2 = NT$ 700
   - 交易 2：手工餅乾 x 5 = NT$ 600
   - 交易 3：手工耳環 x 1 = NT$ 280
   - 交易 4：手工陶杯 x 1 + 手工餅乾 x 3 = NT$ 710
   - 交易 5：手工耳環 x 2 = NT$ 560

## 🐛 常見問題排查

### 問題 1：圖表不顯示

**可能原因：**
- Recharts 未正確安裝
- 數據格式錯誤
- 容器高度為 0

**解決方法：**
1. 檢查 Recharts 安裝：
   ```bash
   npm list recharts
   ```
2. 檢查控制台錯誤訊息
3. 檢查數據格式：
   ```javascript
   console.log('chartData:', chartData);
   ```

### 問題 2：數據不更新

**可能原因：**
- useMemo 依賴項錯誤
- 資料庫查詢失敗
- 日期範圍計算錯誤

**解決方法：**
1. 檢查控制台錯誤
2. 檢查 useMemo 依賴項
3. 檢查資料庫數據：
   ```javascript
   // 在控制台執行
   const db = await indexedDB.open('MarketPulseDB');
   // 檢查 dailyStats 表
   ```

### 問題 3：樣式不正確

**可能原因：**
- Tailwind CSS 未正確編譯
- 類名拼寫錯誤
- 瀏覽器快取

**解決方法：**
1. 重新啟動開發伺服器
2. 清除瀏覽器快取（Ctrl + Shift + R）
3. 檢查 Tailwind 配置

## 🎯 快速驗證腳本

### 檢查數據庫狀態

```javascript
// 在控制台執行
(async function() {
  const db = await new Promise((resolve, reject) => {
    const request = indexedDB.open('MarketPulseDB');
    request.onsuccess = () => resolve(request.result);
    request.onerror = () => reject(request.error);
  });
  
  const tx = db.transaction(['dailyStats'], 'readonly');
  const store = tx.objectStore('dailyStats');
  const stats = await new Promise(resolve => {
    const req = store.getAll();
    req.onsuccess = () => resolve(req.result);
  });
  
  console.log('每日統計數據：', stats);
  console.log('總營收：', stats.reduce((sum, s) => sum + s.revenue, 0));
  console.log('總利潤：', stats.reduce((sum, s) => sum + s.profit, 0));
  console.log('總成交：', stats.reduce((sum, s) => sum + s.dealCount, 0));
  
  db.close();
})();
```

### 檢查商品分類

```javascript
// 在控制台執行
(async function() {
  const db = await new Promise((resolve, reject) => {
    const request = indexedDB.open('MarketPulseDB');
    request.onsuccess = () => resolve(request.result);
    request.onerror = () => reject(request.error);
  });
  
  const tx = db.transaction(['products'], 'readonly');
  const store = tx.objectStore('products');
  const products = await new Promise(resolve => {
    const req = store.getAll();
    req.onsuccess = () => resolve(req.result);
  });
  
  const categoryData = products.reduce((acc, p) => {
    const revenue = (p.totalSold || 0) * p.price;
    if (!acc[p.category]) {
      acc[p.category] = { revenue: 0, count: 0 };
    }
    acc[p.category].revenue += revenue;
    acc[p.category].count += (p.totalSold || 0);
    return acc;
  }, {});
  
  console.table(categoryData);
  
  db.close();
})();
```
