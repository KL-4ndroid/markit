## 快速代碼片段

### 卡片容器
```tsx
<div className="bg-white rounded-[1.5rem] p-6 shadow-lg shadow-[#7B9FA6]/10">
  {/* 內容 */}
</div>
```

### 統計卡片
```tsx
<div className="bg-white rounded-[1.25rem] p-4 shadow-md shadow-[#7B9FA6]/5">
  <div className="bg-gradient-to-br from-[#7B9FA6] to-[#7B9FA6]/70 w-10 h-10 rounded-xl flex items-center justify-center">
    <Icon className="w-5 h-5 text-[#3A3A3A]" />
  </div>
  <div className="text-xs text-[#6B6B6B]">標籤</div>
  <div className="text-[#3A3A3A] text-xl font-medium">數值</div>
</div>
```

### 按鈕
```tsx
<button className="bg-white/20 hover:bg-white/30 px-3 py-2 rounded-xl text-sm font-medium transition-colors disabled:opacity-50">
  <Icon className="w-4 h-4" />
  文字
</button>
```

### 圖表容器
```tsx
<div className="h-48">
  <ResponsiveContainer width="100%" height="100%">
    {/* 圖表 */}
  </ResponsiveContainer>
</div>
```
