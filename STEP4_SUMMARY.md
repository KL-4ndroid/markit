## 📊 技術架構

### 資料流程

```
使用者操作
    ↓
React 組件 (ProductCard, AddProductForm, etc.)
    ↓
Hooks (createProduct, updateProduct, deleteProduct)
    ↓
recordEvent() - 記錄事件到 events 表
    ↓
Event Handler - 處理事件邏輯
    ↓
Dexie.js - 更新 IndexedDB (products 表)
    ↓
useLiveQuery - 響應式更新 UI
```

### 技術棧

- **框架：** Next.js 14+ (App Router)
- **語言：** TypeScript
- **資料庫：** Dexie.js (IndexedDB)
- **狀態管理：** useLiveQuery (響應式)
- **樣式：** Tailwind CSS
- **圖標：** Lucide React
- **通知：** Sonner (Toast)
