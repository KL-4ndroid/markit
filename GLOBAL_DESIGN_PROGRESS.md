## 🔧 技術實施

### CSS 變量系統
```css
:root {
  /* 配色 */
  --mist-blue: #7B9FA6;
  --warm-wood: #D4A574;
  --soft-pink: #F5E6E8;
  --soft-green: #E8F3E8;
  --soft-yellow: #FFF8E7;
  
  /* 圓角 */
  --radius-lg: 1.5rem;
  --radius-md: 1.25rem;
  --radius-sm: 0.75rem;
}
```

### 自定義樣式類
```css
.card-japanese { /* 日系卡片 */ }
.gradient-header { /* 漸變頭部 */ }
.stat-card { /* 統計卡片 */ }
.input-japanese { /* 日系輸入框 */ }
```

### Tailwind 工具類
```tsx
// 大卡片

// 小卡片
className="bg-white rounded-[1.25rem] shadow-md shadow-[#7B9FA6]/5"

// 按鈕
```
