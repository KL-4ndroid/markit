## 📊 核心數據區域（兩種模式都會顯示）

### 4. 市集總覽卡片 ⭐
```
┌─────────────────────────────────────┐
│ 🟢 本場市集：值得再來                │
│                                     │
│ 健康分數: 85.0 / 100  💡            │
│                                     │
│ ┌─────┐ ┌─────┐ ┌─────┐            │
│ │人流  │ │成交  │ │客單  │            │
│ │品質  │ │效率  │ │價   │            │
│ │精準  │ │30%  │ │$450 │            │
│ │高效  │ │     │ │     │            │
│ └─────┘ └─────┘ └─────┘            │
│                                     │
│ 💡 建議：擴大曝光，增加備貨          │
└─────────────────────────────────────┘
```

**使用者看到的有用資訊**:
1. **健康分數** (0-100分)
   - 一眼看出這個市集好不好
   - 點擊 💡 可以看詳細說明
   
2. **人流品質診斷**
   - 精準高效 / 流量不足 / 轉換不足 / 客單價偏低 / 均衡穩定
   - 告訴你問題在哪裡
   
3. **成交效率**
   - 30% = 每10個客人有3個買單
   - 直觀理解轉換率
   
4. **客單價**
   - 平均每筆訂單金額
   - 知道客人願意花多少錢
   
5. **具體建議**
   - 不只告訴你問題，還告訴你怎麼改善

### 5. 核心 KPI 卡片
```
┌─────────────────────────────────────┐
│ 平均成交率: 25%                      │
│ 商品親和力: 手工皂 + 護唇膏 (2.5x)   │
└─────────────────────────────────────┘
```

**使用者看到的有用資訊**:
- 整體平均成交率（所有市集的平均）
- 最強的商品組合推薦

```
┌─────────────────────────────────────┐
```

```
│ healthScore = 70 + weightedScore×15 │
│                                     │
│ 📝 範例                             │
│ 如果各項指標都是平均水準，評分會是   │
│ 70分。如果都優於平均1個標準差，      │
│ 評分會是85分（S級）。                │
│                                     │
│ 🎯 如何解讀                         │
│ S級(≥85): 金牌市集，強烈推薦        │

│ B級(≥60): 穩定市集，可以考慮        │
```

## 🎨 視覺設計特色

### 顏色系統
使用者可以透過顏色快速判斷好壞：

- 🟢 **綠色** - 表現好、向上趨勢
- 🔴 **紅色** - 表現差、向下趨勢
- 🟡 **黃色/金色** - S級、最高等級
- 🔵 **藍色** - 中等水準
- 🟠 **橙色** - 需要注意

### 圖示系統
每個指標都有對應的圖示，讓使用者一眼識別：

- 🌟 健康評分
- ⏰ 時薪
- 💰 成交率
- 💵 客單價
- 🎪 攤位費回收
- 📊 數據可靠度
- 🚶 流量不足
- 👀 轉換不足
- 🛒 客單價偏低
- 🎯 精準高效
- ⚖️ 均衡穩定

## 🎯 使用者能做的決策

基於這些資訊，使用者可以做出以下決策：

### 1. 市集選擇決策
- ✅ 知道哪個市集最值得再去（健康評分排行）
- ✅ 知道哪個市集最賺錢（最有價值市集）
- ✅ 知道哪個市集客群品質好（客單價排行）

- ✅ 看診斷處方箋，知道要改善什麼
- ✅ 看象限分類，知道該用什麼策略

### 3. 商品策略調整
- ✅ 知道哪些商品最暢銷（商品排行）
- ✅ 知道哪些商品該做組合（商品關聯）

### 4. 定價策略
- ✅ 看客單價，知道客人願意花多少錢
- ✅ 看不同市集的客單價差異
- ✅ 調整定價策略

### 5. 時間管理
- ✅ 看時薪，評估時間投資報酬率
- ✅ 看營業時數，決定是否值得
- ✅ 看每日趨勢，選擇最佳時段

## 📱 使用流程範例

### 情境 1: 快速查看本月表現
1. 進入分析頁面（預設本月）
2. 看「市集總覽卡片」- 知道整體表現
3. 看「核心 KPI」- 了解關鍵數據
4. **30秒內完成** ⚡

### 情境 2: 決定下次要去哪個市集
1. 切換到「進階模式」
2. 看「市集綜合評分排行榜」- 找出最高分市集
3. 看「最有價值市集」- 確認最賺錢的
4. 看「象限網格」- 了解市集類型
5. **做出決策：選擇評分高且賺錢的市集** ✅

### 情境 3: 優化經營策略
1. 看「市集總覽」的診斷結果
2. 例如：「轉換不足 - 人很多但不買單」
3. 看處方箋的3個建議：
   - 改善銷售話術
   - 調整定價策略
   - 增加購買誘因
4. 點擊 💡 看詳細說明
5. **下次市集執行改善方案** 🎯

### 情境 4: 提升客單價
1. 看「商品關聯分析」
2. 發現：手工皂 + 護唇膏 (2.5x關聯)
3. 做成組合包優惠
4. 擺在一起展示
5. **下次市集客單價提升** 💰
