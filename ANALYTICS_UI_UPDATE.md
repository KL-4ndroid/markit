# 分析頁面 UI/UX 更新說明

## 更新日期
2025年12月29日

## 更新概述
根據 MarketApp 專案的設計系統，完全重新設計了分析頁面的 UI/UX，採用日系文創風格。

## 主要變更

### 1. 配色方案（日系文創色彩）
- **霧藍 (Mist Blue)**: `#7B9FA6` - 主色調
- **暖木 (Warm Wood)**: `#D4A574` - ���要色調
- **柔粉 (Soft Pink)**: `#F5E6E8` - 輔助色
- **柔綠 (Soft Green)**: `#E8F3E8` - 輔助色
- **柔黃 (Soft Yellow)**: `#FFF8E7` - 輔助色
- **背景色**: `#FAFAF8` - 溫暖的米白色
- **文字色**: `#3A3A3A` - 深灰色
- **次要文字**: `#6B6B6B` - 中灰色

### 2. 設計元素更新

#### 圓角設計
- 卡片圓角：`1.5rem` (24px)
- 小卡片圓角：`1.25rem` (20px)
- 按鈕圓角：`0.75rem` (12px)
- 圖表柱狀圓角：`12px` (頂部)

#### 陰影效果
- 使用柔和的陰影：`shadow-lg shadow-[#7B9FA6]/10`
- 小卡片陰影：`shadow-md shadow-[#7B9FA6]/5`

#### 漸變效果
- 頂部標題區域：`from-[#7B9FA6] to-[#D4A574]`
- 統計卡片圖標背景：使用各種配色的漸變

### 3. 組件更新

#### 頂部標題區域
- 使用漸變色背景（霧藍到暖木）
- 圓角底部設計 `rounded-b-[2rem]`
- 添加返回按鈕（使用 lucide-react 圖標）
- 導出按鈕改用圖標 + 文字組合

#### 統計卡片（4個）
- 市集總數（Package 圖標）
- 總收入（DollarSign 圖標）
- 總淨利潤（TrendingUp 圖標）
- 平均轉換率（Users 圖標）
- 每個卡片都有對應的漸變色圖標背景

#### 收入趨勢圖表
- 從折線圖改為面積圖（AreaChart）
- 使用漸變填充效果
- 霧藍色代表收入，暖木色代表利潤
- 添加圖例說明

#### 轉換漏斗
- 簡化為兩個階段：總互動 → 成交數
- 使用橫向進度條設計
- 顯示百分比和實際數值
- 底部顯示整體轉換率

#### 互動與成交對比圖表
- 使用柱狀圖（BarChart）
- 圓角柱狀設計
- 霧藍色代表互動數，柔綠色代表成交數

#### 成本分析卡片
- 簡潔的三行設計
- 使用品牌色彩標示不同數據
- 清晰的分隔線

#### 市集明細列表
- 圓角卡片設計
- 柔和的邊框顏色
- hover 效果使用背景色變化

### 4. 圖標更新
使用 `lucide-react` 圖標庫替代 emoji：
- `ArrowLeft` - 返回按鈕
- `Download` - 導出按鈕

- `DollarSign` - 金額圖標
- `Users` - 用戶圖標
- `Package` - 包裹圖標

#### 配色統一
- 所有圖表使用統一的品牌色彩
- 網格線使用 `#7B9FA6` 並設置透明度

#### Tooltip 樣式
- 白色背景
- 柔和的邊框
- 圓角設計 `12px`
- 統一的字體大小

#### 圖表漸變
- 面積圖使用漸變填充
- 從色彩到透明的自然過渡

### 6. 響應式設計
- 最大寬度限制：`max-w-lg` (32rem)
- 統計卡片使用 2 列網格布局
- 適配移動端顯示

### 7. 載入狀態
- 使用品牌色的載入動畫
- 背景色與整體設計一致

## 技術細節

### 依賴項
- `recharts` - 圖表庫
- `lucide-react` - 圖標庫
- `next/link` - 路由導航

### 顏色變量
建議在 `globals.css` 或 `tailwind.config.ts` 中定義這些顏色變量，以便全局使用：

```css
:root {
  --mist-blue: #7B9FA6;
  --warm-wood: #D4A574;
  --soft-pink: #F5E6E8;
  --soft-green: #E8F3E8;
  --soft-yellow: #FFF8E7;
  --background: #FAFAF8;
  --foreground: #3A3A3A;
  --muted-foreground: #6B6B6B;
}
```
