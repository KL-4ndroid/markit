## 🔄 完整的用戶流程

### 匿名用戶流程

1. **打開應用程式**
   - 看到頂部導航欄
   - 顯示「登入」按鈕
   - 可以正常使用所有功能（離線優先）

2. **建立市集和商品**
   - 資料儲存在本地 Dexie
   - owner_id 為 'local'
   - sync_status 為 'local_only'

3. **點擊登入按鈕**
   - 彈出 LoginModal
   - 輸入 Email 和密碼
   - 或使用 Magic Link

4. **登入成功**
   - 系統自動檢測本地資料
   - 發現有 X 個市集和 Y 個事件
   - 彈出 MigrationModal

5. **選擇遷移選項**
   
   **選項 A：確認同步**
   - 顯示「處理中...」
   - 更新所有資料的 owner_id
   - 標記為待同步
   - 背景同步引擎自動上傳
   - 顯示「完成！」
   - 關閉對話框

   **選項 B：清除並登入**
   - 顯示「處理中...」
   - 清空本地 Dexie
   - 從雲端拉取資料
   - 重放事件重建快照
   - 顯示「完成！」
   - 關閉對話框

   **選項 C：取消登入**
   - 登出帳號
   - 保留本地資料
   - 關閉對話框

6. **登入後使用**
   - 頂部顯示用戶 Email
   - 顯示同步狀態
   - 可以手動觸發同步
   - 離線時自動進入離線模式

---

### 已登入用戶流程

1. **打開應用程式**
   - 自動恢復登入狀態
   - 顯示用戶資訊
   - 顯示同步狀態

2. **背景自動同步**
   - 每 30 秒自動同步
   - 網路恢復時自動同步
   - 顯示同步進度

3. **查看同步狀態**
   - Hover 同步狀態按鈕
   - 查看詳細資訊
   - 手動觸發同步

4. **登出**
   - 點擊用戶頭像
   - 選擇「登出」
   - 本地資料保留

## 📝 使用說明

### 如何觸發登入

**方法 1：點擊頂部導航欄的「登入」按鈕**

**方法 2：程式碼觸發**
```typescript
import { triggerLogin } from '@/components/auth/AuthManager';

// 在任何地方調用
triggerLogin();
```

### 如何檢查登入狀態

```typescript
import { useAuth } from '@/lib/supabase/auth-context';

function MyComponent() {
  const { user, loading } = useAuth();
  
  if (loading) return <div>載入中...</div>;
  if (!user) return <div>未登入</div>;
  
  return <div>歡迎，{user.email}</div>;
}
```

### 如何使用同步功能

```typescript
import { useSync } from '@/hooks/useSync';

function MyComponent() {
  const { status, sync, pendingCount } = useSync();
  
  return (
    <div>
      <p>狀態：{status}</p>
      <p>待同步：{pendingCount} 個事件</p>
      <button onClick={sync}>手動同步</button>
    </div>
  );
}
```
