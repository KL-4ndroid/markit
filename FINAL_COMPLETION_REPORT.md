| 元素類型 | 圓角大小 | CSS 類名 | 用途 |

| 元素類型 | 陰影樣式 | 效果描述 |

| Hover | shadow-xl shadow-[#7B9FA6]/15 | 懸停加深效果 |

### 字體規範
