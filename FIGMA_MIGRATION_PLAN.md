## ✅ 可行性分析

### 技術相容性
- ✅ 兩個專案都使用 React + TypeScript
- ✅ 都使用 Tailwind CSS
- ✅ 都使用 lucide-react 圖標庫
- ⚠️ 架構差異: MarketApp (Vite/SPA) vs 當前專案 (Next.js/SSR)
- ✅ 可以提取純 UI 樣式，不影響功能邏輯

### 設計元素對比

| 設計元素 | MarketApp | 當前專案 | 遷移難度 |
|---------|-----------|---------|---------|
| 色彩系統 | 柔和色調 (#7B9FA6, #D4A574) | 標準藍色系 | ⭐ 簡單 |

| 卡片樣式 | 漸層陰影、柔和背景 | 基礎白卡 | ⭐⭐ 中等 |
| 導航欄 | 底部固定、圖標+文字 | 底部固定、圖標+文字 | ⭐ 簡單 |
| 頭部設計 | 漸層背景、圓角底部 | 純色背景 | ⭐⭐ 中等 |
| 互動效果 | hover/active 動畫 | 基礎互動 | ⭐⭐ 中等 |

## 🎯 核心設計元素

### 1. 色彩系統
```css
/* 主色調 */
--sage-green: #7B9FA6;      /* 鼠尾草綠 - 主色 */
--warm-sand: #D4A574;        /* 暖沙色 - 輔助色 */
--soft-pink: #F5E6E8;        /* 柔粉色 - 背景 */
--light-yellow: #FFF8E7;     /* 淺黃色 - 強調 */
--mint-green: #E8F3E8;       /* 薄荷綠 - 成功 */
--warm-white: #FAFAF8;       /* 暖白色 - 頁面背景 */

/* 文字色 */
--text-primary: #3A3A3A;     /* 主文字 */
--text-secondary: #6B6B6B;   /* 次要文字 */
```

### 2. 圓角系統
- 小圓角: `rounded-xl` (0.75rem)
- 中圓角: `rounded-[1.25rem]` (1.25rem)
- 大圓角: `rounded-[1.5rem]` (1.5rem)
- 超大圓角: `rounded-2xl` (2rem)

### 3. 陰影系統
- 卡片陰影: `shadow-lg shadow-[#7B9FA6]/10`
- 懸浮陰影: `hover:shadow-xl`
- 導航陰影: `shadow-lg`

### 4. 漸層效果
- 頭部漸層: `from-[#7B9FA6] to-[#D4A574]`
- 按鈕漸層: `from-[#F5E6E8] to-[#F5E6E8]/70`
